# An (heavily WIP) open-source Gelbooru client
At least it works

## Dependencies:
[Node](https://nodejs.org)
[Git](https://git-scm.com)

## Installation:
```
git clone https://github.com/tzwel/gelbooru_client.git && cd gelbooru_client && npm install
```

#### Running the client:
Running start.cmd or

```
npm start
```
